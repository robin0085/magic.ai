### 💡游戏动画：起势动作

起势是主动作的预备动作，
是大跳跃前的小蹲伏，
是击球前的向后举棒。

起势的重要功能，
有助于表现动作的质量感与力量感。

起势能够传递力量，
动作的力量需求越大，起势就越要夸张。

起势还承担着另一个重要的目的，
使动作更为清晰可辨。

起势动作有助于引导观众的视线，
让观众对即将发生的事情做好准备。
极为完善的起势动作，
识别即将到来的攻击，足够快地闪避乃至反击。
